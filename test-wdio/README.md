# Andrew Polovets WebDriverIO+Mocha for UI+API

### Key Artifacts:
* ./.env
* ./tests

### Preconditions:
* Optionally configure .env file with `SPEC_NAMES` without extension separated by comma
  * `SPEC_NAMES=first,second`

### Commands:
* To run UI spec/s:
    * npm test

* To run API spec/s:
  * npm run testApi
